## Install

Overwrite Ayu File Location

Windows
```shell
C:\Users\%USERPROFILE%\.vscode\extensions\teabyii.ayu-1.0.5
```
MacOS
```shell
~/.vscode/extensionsteabyii.ayu-1.0.5
```
Linux
```shell
~/.vscode/extensions/teabyii.ayu-1.0.5
```
